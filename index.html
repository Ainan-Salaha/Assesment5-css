What are CSS combinators

A combinator is that explains the relationship between the selectors.
-We can add a combinator in a simple selector and css selector can contain more than one selector.

    Types of Combinators

  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)
